UART top design (transmitter, receiver, baudrate generator, register file). Sync active-high `rst`. Memory-mapped register writes via `address[1:0]` + `write_data[31:0]` + `we` (set baud / control / TX data); reads via `re` produce `read_data[7:0]`. Serial `tx` is driven by the TX FSM, `rx` is the line input.

`context/starting_point.v` provides the verbatim DR_RTL baseline as a starting point for optimization.

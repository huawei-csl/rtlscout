Tiny FSM-style CPU with an 8-deep 16-bit instruction memory and a 4 × 8-bit register file. Sync active-high `rst` initialises PC=0, state=FETCH, and the register file to {0x11, 0x22, 0x33, 0x44}. `en` advances the fetch/decode/exec FSM; ports `imem_we / imem_addr / imem_wdata` write the program memory and `rf_we / rf_addr / rf_wdata` write the register file. Debug outputs `PC`, `halt`, `dbg_r0..r3`, and `dbg_state` are sampled every cycle by the testbench.

`context/starting_point.v` provides the verbatim DR_RTL baseline as a starting point for optimization.

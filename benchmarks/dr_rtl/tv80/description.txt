TV80 — synthesizable Z80 CPU core (with 8080 / GB modes via parameters). Sync active-low `reset_n`. `cen` is the clock enable; `int_n`, `nmi_n`, `busrq_n`, `wait_n` are interrupt / bus / wait control signals; `dinst[7:0]` and `di[7:0]` are instruction-bus and data-bus inputs respectively. Outputs are the full Z80 bus signal set (`m1_n`, `iorq`, `rfsh_n`, `halt_n`, `busak_n`, `A[15:0]`, `d_o[7:0]`, …) plus tracing fields `mc[2:0]`/`ts[2:0]`/`intcycle_n`/`IntE`/`stop`.

`context/starting_point.v` provides the verbatim DR_RTL baseline as a starting point for optimization.

Design a module named seq_det that detects the serial bit pattern 1-0-1-1 on input bit in (most recent bit is last). The module has inputs clk, rst_n, in, and output match. It should assert match=1 for exactly one cycle when the sequence 1011 is observed, allowing overlapping matches. Use a synchronous active-low reset: on a rising clock edge, if rst_n is 0 then reset state and match must be 0.
